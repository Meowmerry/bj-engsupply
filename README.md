
# B&J Engineering Website

Company website for **B&J ENGINEERING AND SUPPLY CO.,LTD.** built with React, TypeScript, and Tailwind CSS.

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Start Development Server**
    ```bash
    npm run dev
    ```
    Open your browser to the URL shown (usually `http://localhost:5173`).

3.  **Build for Production**
    ```bash
    npm run build
    ```
    The output will be in the `dist` folder.

## Project Structure

-   `src/components`: UI Components (Navbar, Hero, Services, etc.)
-   `src/data`: Static data (Services list, Company info)
-   `src/App.tsx`: Main application entry point
